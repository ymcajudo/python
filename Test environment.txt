1. 내 컴 환경
- Ubuntu 18.04 서버 on VMware: 10.0.2.15
- VMware의 호스트 IP: 192.168.10.1
- 호스트 IP에서 Ubuntu 서버로 NAT/포트포워딩 설정
 .192.168.10.1(22) → 10.0.2.15(22)			#SSH 접속
 .192.168.10.1(65432) → 10.0.2.15(65432)		#tcp 소켓 서버용 포트(65432)

2.리눅스 서버에 소켓 프로그램(서버) 설치시
- python3.10 설치
- 소켓 프로그램 실행
 .~#python3.10 ${test.py}
 .서버IP는 Ubuntu 서버의 IP로 설정(10.0.2.15)
 .포트는 65432

3.소켓 클라이언트(Hercules 사용) 설정
-접속IP: VMware의 호스트 IP 입력(192.168.10.1)
-포트: 65432
-connect 버튼 클릭
